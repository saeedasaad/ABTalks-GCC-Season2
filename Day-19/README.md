### Day-19: Min Stack (Software Engineering - Stack Design)

- **Problem Statement:**  
  Design a stack that supports the following operations efficiently:  
  - `push(x)`: Add element `x` onto the stack.  
  - `pop()`: Remove the element on top of the stack.  
  - `top()`: Retrieve the element on top of the stack.  
  - `getMin()`: Retrieve the minimum element in the stack.  

- **Challenge Overview:**  
  - Extend the basic stack data structure with additional functionality.  
  - Ensure that retrieving the minimum element is efficient.  
  - Optimize operations so they run in constant time.  
  - Explore how auxiliary data structures can be combined with stacks for real-world use cases.  

- **Topics Covered:**  
  - Advanced stack concepts  
  - Designing efficient data structures  
  - Constant-time operations (`O(1)`)  
  - Problem-solving techniques for interviews  
  - Clean and structured coding practices
