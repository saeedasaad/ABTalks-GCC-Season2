## Day-50: NumPy Basics (Vectors & Loss)

- **Problem Statement:**  
  Implement statistical operations using NumPy to calculate the **mean** and **variance** of a given dataset.

- **Challenge Overview:**  
  - This challenge introduces **NumPy fundamentals** for efficient numerical computing.  
  - Participants will work with arrays and implement **vector operations**.  
  - Mean and variance are essential statistical measures used in **data analysis** and **machine learning**.  
  - Understanding these operations builds the foundation for **loss functions** in ML models.  

- **Topics Covered:**  
  - NumPy arrays  
  - Vector operations  
  - Mean & variance  
  - Data understanding  
  - ML basics  

- **Solution File:**  
  `solution.py`  

- **Approach:**  
  1. Convert the input list into a NumPy array.  
  2. Use `np.mean()` to calculate the average value.  
  3. Use `np.var()` to calculate the variance.  
  4. Return both values for further use in ML or statistical analysis.  

- **Time & Space Complexity:**  
  - **Time Complexity:** `O(n)` (each element is processed once)  
  - **Space Complexity:** `O(n)` (array storage in memory)
