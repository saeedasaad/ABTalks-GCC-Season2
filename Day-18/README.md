### Day-18: Valid Parentheses (Software Engineering - Stack Basics)

- **Problem Statement:**  
  Given a string containing just the characters `'('`, `')'`, `'{'`, `'}'`, `'['`, and `']'`, determine if the input string is valid.  
  A string is valid if:  
  - Open brackets are closed by the same type of brackets.  
  - Open brackets are closed in the correct order.  

- **Challenge Overview:**  
  - Work with a string containing different types of parentheses.  
  - Understand how stack-based logic helps in validation problems.  
  - Use list operations to simulate stack behavior.  
  - Determine whether the given sequence of parentheses is valid.  

- **Topics Covered:**  
  - Stack data structure fundamentals  
  - Using lists as stacks in Python  
  - Push and pop operations  
  - Validation and parsing problems  
  - Clean and structured coding practices
